Features of Amebot
We provide you the best features in Amebot!
Feature #1

Amebot is always online on the server to best serve you. If you need Amebot, it'll be there to serve you right away. Amebot is always accompanies you wherever you are. Our motto is fast and accurate

Feature #2

Amebot is very easy to use, you just add it to your server discord and enter command /play with your link youtube to listen to music. If you want to enter more commands, you can type /help to show all commands and use them.

Amebot strictly secures users' personal information and does not disclose information to the outside. If you find your information leaked, please report it, we will fix it immediately. It is Amebot's honor to serve you.
